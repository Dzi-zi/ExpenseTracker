Smart Expense Tracker

A full-stack expense tracking application built with React, Node.js, Express, and MongoDB. Track your expenses, visualize spending patterns, and export data with a beautiful, responsive interface.

 Features

- Real-time Expense Tracking: Add, edit, and delete expenses instantly
- Visual Analytics: View spending by category with interactive charts
- Monthly Comparisons: Track spending trends month-over-month
- Category Filtering: Filter expenses by customizable categories
- CSV Export: Download expense data for external analysis
- Responsive Design: Works seamlessly on mobile, tablet, and desktop
- RESTful API: Clean, documented backend API
